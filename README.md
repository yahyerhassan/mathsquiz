# mathsquiz

**My First Maths Quiz**

We know you've had a long tiring week but I know you are still up for a challenge!!

How to run the quiz

1. Make sure you have a recent updated python version on your computer
2. Clone this repository into your terminal

Yes it is that simple. Good luck! You might need it.

What to expect:

1. 10 randomly generated arithmetic question! Yes ONLY 10!
2. Final score out of 10 
3. Be careful, there might be decimals so don't forget to round to 2 decimal points.


Built With Python 3: random module, functions and modular code
